# FIREWALLAUT
Un script que busca automatizar el uso de los 3 firrewalls mas conocidos para linux, ufw, firewalld, iptables

# Instalacion

```bash
git clone https://github.com/JJoosh/FIREWALLAUT.git
cd FIREWALLAUT
chmod +x firwallaut.py
sudo ./firwallaut.py 
```
